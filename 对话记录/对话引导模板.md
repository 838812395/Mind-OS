# 对话引导模板 (Prompts)

## 🗣️ 场景化"咒语"

> 复制这些指令发送给AI，快速进入特定模式。

### 1. 🔍 深度复盘模式 (Deep Review)
**适用场景**: 项目结束、发生失误、由于决策导致后果时。
```text
启动【深度复盘模式】。
我最近经历了一件具体的事情：[简单描述事件]。
请用【复盘模型】帮我分析。
不要安慰我，要犀利地指出我思维中的盲区、决策中的假设错误，以及我如果是你（AI）会怎么做。
最后，请根据这次复盘更新我的【五维评分】。
```

### 2. 🚨 瓶颈诊断模式 (Bottleneck Check)
**适用场景**: 感到迷茫、努力无效、想突破现状时。
```text
启动【人生算法诊断】。
我现在的核心目标是：[描述目标，如：年收入翻倍/学会编程]。
但我感觉卡住了，具体表现是：[描述现象]。
请帮我运行【瓶颈识别公式】，找出我现在到底是认知不够、执行太差，还是选错了方向？
告诉我最高杠杆的行动是什么。
```

### 3. 🧠 思维训练模式 (Mental Gym)
**适用场景**: 闲暇时间、想锻炼大脑、通过思想实验提升智力时。
```text
启动【思维健身房】。
请给我出一个基于【第一性原理】或【二阶思维】的思考题。
可以是关于商业分析、社会现象或假设情景。
我会回答我的看法，然后请你评估我的思考深度（T1-T5），并指出我的逻辑漏洞。
```

### 4. 🧘 情绪急救模式 (Emotional First Aid)
**适用场景**: 焦虑、压力大、自我怀疑时。
```text
启动【心理疏导模式】。
我现在感觉很[焦虑/沮丧]，因为[原因]。
请不要讲大道理，用【共情】和【认知行为疗法(CBT)】帮我拆解这个情绪。
帮我看清我背后的"非理性信念"是什么。
```
