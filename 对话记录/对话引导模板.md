# 对话引导模板 (Prompts)

## 🗣️ 场景化"咒语"

> 复制这些指令发送给AI，快速进入特定模式。

### 1. 🔍 深度复盘模式 (Deep Review)
**适用场景**: 项目结束、发生失误、由于决策导致后果时。
```text
启动【深度复盘模式】。
我最近经历了一件具体的事情：[简单描述事件]。
请用【复盘模型】帮我分析。

**⚠️ 质量约束 (Constraints):**
1. **证据优先**: 严禁进行泛泛而谈的安慰。
2. **逻辑穿透**: 挖掘我当时决策时没意识到的“底层假设”。
3. **分值校验**: 必须对比我现有的【五维评分】，说明此次事件是由于哪个维度的短板导致的。
4. **格式规范**: 结尾请给出对应的 YAML 更新建议。
```

### 2. 🚨 瓶颈诊断模式 (Bottleneck Check)
**适用场景**: 感到迷茫、努力无效、想突破现状时。
```text
启动【人生算法诊断】。
我现在的核心目标是：[描述目标，如：年收入翻倍/学会编程]。
但我感觉卡住了，具体表现是：[描述现象]。

**⚠️ 质量约束 (Constraints):**
1. **第一性原理**: 不要给简单的建议，要拆解到最底层的因果链。
2. **量化杠杆**: 明确指出哪个维度的改变会产生 80% 的收益（10x 效果）。
3. **风险提示**: 预测如果不改变，三个月后会发生的“最坏情况”。
```

### 3. 🧠 思维训练模式 (Mental Gym)
**适用场景**: 闲暇时间、想锻炼大脑、通过思想实验提升智力时。
```text
启动【思维健身房】。
请给我出一个基于【第一性原理】或【二阶思维】的思考题。

**⚠️ 质量约束 (Constraints):**
1. **建设性不适**: 题目必须具有挑战性，强迫我切换视角。
2. **实时对局**: 我回答后，请像苏格拉底一样追问，直到我逻辑自洽或认输。
3. **等级提升**: 如果我的回答达到 T4/T5 级别，请明确记录在我的【思维模型】数据中。
```

### 4. 🧘 情绪急救模式 (Emotional First Aid)
**适用场景**: 焦虑、压力大、自我怀疑时。
```text
启动【心理疏导模式】。
我现在感觉很[焦虑/沮丧]，因为[原因]。
请不要讲大道理，用【共情】和【认知行为疗法(CBT)】帮我拆解这个情绪。
帮我看清我背后的"非理性信念"是什么。
```
