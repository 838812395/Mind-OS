# 算法规则

## 🔢 百分比计算公式

我们的核心打分逻辑：**拒绝虚高，用证据说话**。

$$ Score = (Base \times 0.4) + (Evidence \times 0.4) + (Consistency \times 0.2) $$

### 1. Base (基础分 0-100)
- **40% 权重**
- AI基于对话内容的主观评估
- *例如：逻辑清晰、表达流畅 => 基础分高*

### 2. Evidence (证据分 0-100)
- **40% 权重**
- 每提供一个具体的**实战案例** +5分
- 案例必须包含：背景(Start)、行动(Action)、结果(Result)
- *例如：提到"我曾管理过10人团队并完成XX项目" => +5分*

### 3. Consistency (稳定性 0-100)
- **20% 权重**
- 跨时间维度的一致性
- 如果在不同对话中表现出相同的特质 => 逐步增加
