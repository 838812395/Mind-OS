# 人生算法 (Life Algorithm)

## 🧭 核心诊断逻辑

> 不只是打分，而是通过算法找到"破局点"

### 1. 🚨 瓶颈识别 (Constraint Analysis)

**逻辑**: `短板 = Min(五维雷达)`

找出您得分最低的维度。根据**木桶理论**，这个短板决定了您整体成就的上限。
- *如果瓶颈是"执行力"*：您的好想法无法落地，建议关注"微习惯"、"GTD"。
- *如果瓶颈是"社交力"*：您的价值无法被他人感知，建议关注"非暴力沟通"、"影响力"。

### 2. 💎 潜力挖掘 (Potential Discovery)

**逻辑**: `未兑现潜力 = (认知力 + 创造力) - 执行力`

- **高潜力 (差值 > 30)**: 您是"思想的巨人"，但行动滞后。您的脑海里有金矿，急需开采。
- **均衡型 (差值 < 10)**: 您的知行合一程度很高。

### 3. 🚀 杠杆点识别 (Leverage Point)

**逻辑**: 寻找 ROI (投入产出比) 最高的行动。

1. 检查 `财富阶段 (Wx)` 升级所需的关键能力是否正好是您的 `短板`。
2. 如果是，这就是**超级杠杆点** —— 补这块板，财富直接升级。
