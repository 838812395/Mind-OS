# 潜意识重塑 (Subconscious Rewiring)

## 🧠 认知重构 (Cognitive Restructuring - CBT)

> "你感受到的不是事情本身，而是你对事情的看法。" —— Epictetus
> **原理**: 通过理性证据，重写限制性信念。

| 限制性信念 (Old Narrative) | 触发情绪 (Trigger) | 挑战证据 (Disputing) | 新神经回路 (New Narrative) |
|--------------------------|-------------------|----------------------|---------------------------|
| *例如：我只要犯错就会被大家看不起。* | 恐惧/焦虑 | *1. 乔帮主被开除过。2. 上次我犯错大家反而帮我。* | **犯错是成长的必修课，大家的关注点不在我身上。** |
| - | - | - | - |

## ⚡ 执行意图 (Implementation Intentions - Gollwitzer)

> **原理**: 将控制权交给环境线索，实现"战略性自动化" (Strategic Automaticity)。
> **公式**: If [情境] Then [行动]

### 自动化脚本库

- [ ] **防御焦虑**:
    - **IF** 我感到胸口发紧 (焦虑来袭)，
    - **THEN** 我立刻深呼吸3次并问自己"最坏的结果是什么？"。
- [ ] **对抗拖延**:
    - **IF** 我想拿起手机刷视频，
    - **THEN** 我立刻把手机扔到沙发对面，并数到5马上打开书。
- [ ] **自定义脚本**:
    - **IF** ______
    - **THEN** ______
